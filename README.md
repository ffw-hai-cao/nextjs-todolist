## Getting Started

First, run the development server:

```bash
# Install project
npm install
# Run the project on local
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

If the port not runing, try:
```bash
npx kill-port 3000
```
